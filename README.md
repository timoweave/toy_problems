# Install and checkout the result
---
run `npm test` to see the reports.html loaded into browser (thanks `mochawesome` module).

1. `npm install`
1. `npm test`
1. `npm run test_mocha`
1. `npm run test_jasmine`
1. `npm run test_karma`
1. `npm run test_phantom`
